## Enforcement

The `VersionEnforcer` hook (SessionStart) will warn if a project's dependencies fall below these minimums:

```typescript
const MIN_VERSIONS = {
  bun: '1.3.0',
  typescript: '5.9.0',
  zod: '4.0.0',
  biome: '2.3.0',
  effect: '3.19.0',
};
```

## Version Sources

```
lib/versions.nix          # Single source of truth (Nix)
    ↓
config/signet/versions.json   # Generated on home-manager activation
    ↓
Signet generators         # Read versions.json for scaffolding
```

To update versions:
1. Edit `lib/versions.nix`
2. Run `just rebuild` (regenerates versions.json)
3. Test with `signet validate`
