#  dotfiles
## (1) clone dotfiles into (~/dotfiles)

## (3) Setup

Symlinks needs setting up

```shell
~/.zshrc -> ~/dotfiles/home/zsh/zshrc
~/.zshenv -> ~/dotfiles/home/zsh/zshenv
~/.zprofile -> ~/dotfiles/home/zsh/zprofile
~/.config/zsh -> ~/dotfiles/home/zsh
```

## Run init.sh


```shell
~/dotfiles/init.sh
```
