# Explicit Evolution Cycle

Trigger a deliberate evolution cycle:

## Instructions

1. **Reflect on Recent Tasks**
   - What patterns emerged?
   - What worked well?
   - What caused friction?

2. **Extract Learnings**
   For each insight, determine scope:
   - task: Temporary, don't persist
   - file: Add to file header
   - project: Add to .claude/CLAUDE.md
   - stack: Add to stack template
   - global: Requires validation (propose only)

3. **Update Memory**
   - Add new learnings with timestamps
   - Update confidence scores on existing entries
   - Remove contradicted entries

4. **Refine Patterns**
   - Identify strategy improvements
   - Update workflow preferences
   - Optimize tool usage

5. **Report Evolution**
   ```
   Evolution Cycle Complete
   ─────────────────────────
   New learnings: X
   Updated entries: Y
   Removed entries: Z
   Proposed global changes: N (pending validation)
   ```
