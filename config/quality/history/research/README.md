# Research Notes

This directory stores research summaries from investigation sessions.

## Format

Each research topic is stored as `{date}-{topic}.md`:

```markdown
# Research: {Topic}

## Date
2025-12-17

## Question
What question prompted this research?

## Sources
- URL 1
- URL 2
- Documentation reference

## Findings
1. Key finding one
2. Key finding two
3. Key finding three

## Conclusions
Summary and recommendations.

## Next Steps
- Action item 1
- Action item 2
```

## When to Create Research Notes

- Before implementing a major feature
- When evaluating libraries or tools
- After debugging a complex issue
- When exploring new patterns
