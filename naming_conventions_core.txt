What is PascalCase?	Each word starts with uppercase, no spaces/punctuation<br><br>Examples:<br>• CustomerOrder<br>• UserProfilePicture<br>• HttpRequestHandler<br><br>Used for: Classes in C#/Java, public APIs	naming-conventions
What is camelCase?	First word lowercase, rest start uppercase, no spaces/punctuation<br><br>Examples:<br>• userProfile<br>• httpRequestHandler<br>• customerOrderList<br><br>Used for: Variables/functions in JavaScript, private methods	naming-conventions
What is snake_case?	All lowercase, words separated by underscores<br><br>Examples:<br>• user_profile<br>• http_request_handler<br>• customer_order_list<br><br>Used for: Python variables/functions, SQL tables/columns	naming-conventions
What is kebab-case?	All lowercase, words separated by hyphens<br><br>Examples:<br>• user-profile<br>• http-request-handler<br>• customer-order-list<br><br>Used for: CSS classes/IDs, URL slugs	naming-conventions

What's the key difference between PascalCase and camelCase?	PascalCase: First letter is uppercase (CustomerOrder)<br>camelCase: First letter is lowercase (customerOrder)<br><br>Both have no spaces and capitalize each word after the first	naming-conventions
